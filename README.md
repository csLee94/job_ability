# 취업핵심역량 도출 PROJECT 개요
### Project 주제
코로나 등 경제위기로 취업난이 심해지고 있으며, 공채에서 수시채용으로의 채용 형태 변화는 취준생이 취업을 더 어렵게 느끼게 되었다. 또한 다양한 취업 관련 커뮤니티에서도 주요 취업 요건에 대한 갑론을박이 많아, 취업핵심역량을 도출해보고자 한다.
